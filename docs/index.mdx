# @sinoui/icons

@sinoui/icons 集成了[Material Design](https://material.io/resources/icons/?style=baseline)提供的 1100+个图标，均为 svg 格式图标。
@sinoui/icons 中所有图标的实现，均符合[Material Design | Icon](https://material.io/design/iconography/system-icons.html#)的设计规范。

## 图标使用

首先需要安装依赖

```shell
yarn add @sinoui/icons
```

由于图标组件的内部实现使用 SvgIcon，用法及相关属性与其保持一致，更多详细使用方式可参考[SvgIcon 官方文档]()。

具体使用时，可先查找需要使用的图标，确定图标名称，然后在具体文件中引入使用：

```tsx
import Close from '@sinoui/icons/Close';
```

按照最新的 Material Design 标准，图标分成以下风格：

- filled
- outlined
- rounded
- two-tone
- sharp

每种图标均有这五种风格。

可能不同风格的图标是不一样的，也可能是一样的。默认提供 filled 风格的，不同风格的图标引用方式也不一样的：

```tsx
import Close from '@sinoui/icons/Close';
import CloseOutlined from '@sinoui/icons/CloseOutlined';
import CloseSharp from '@sinoui/icon/CloseSharp';
import CloseRounded from '@sinoui/icons/CloseRounded';
import CloseTowTone from '@sinoui/icons/CloseTwoTone';
```
